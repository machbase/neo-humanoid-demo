# 목표

- 마크베이스 네오 기반의 웹 기반 3차원 휴머노이드 Physical AI 데모를 구축하는 것
- 이를 통해 Physcial AI를 위한 데이터 처리를 마크베이스 네오를 통해 고속으로 하는 마케팅 목적이 큼
- 고객들에게 성능/기능 등의 관점에서 마크베이스 네오의 활용을 통한 데이터 처리에 대한 감동을 주고자 함.


# 데이터
- 관련되는 휴머노이드 로봇을 위한 Physical AI 의 멋진 데이터 셋을 검색하고 나에게 후보 선택을 물어봐.
- 이를 다운로드하여 마크베이스에서 활용하도록 구현
- 데이터는 휴머노이드 시뮬레이션 및 관련 데모를 잘 활용할 수 있는 대규모 데이터 셋
- 전체 처리 시간은 최소한 10분 정도의 다양한 센서 데이터가 되어야 함


# 웹 데모
 - Web 기반의 데모로서 마크베이스 네오에서 실시간으로 데이터를 추출하여, 시각화하여야 함
 - 끊기지 않고  데이터를 실제 가상 공간에서 자유롭게 시간축으로 오가면서 로봇의 동작 변화를 볼 수 있어야 함.
 - 내부 구조는 가장 빠르고, 예쁜 웹 JS 라이브러리를 활용하도록 한다.
- 화면에는 로봇의 움직이는 모습 동작과 거기에 관련되는 센서의 값이 출력되고, 시간을 네비게이션 가능해야 함.

# 소스코드
 - 사용자가 , 데이터  다운로드 및 앱 구동 데모 실행의 모든 과정에 README.md에 있어야 함.
 - 샘플 소스는 현재코드를 기준으로 JSH 기반으로 구현하고, 현재 디렉토리의 코드를 참고해라.
 - 현재 코드를 샘플로서 최종적으로 불필요한 코드를 수정하고, 제거해라.


# 마크베이스 네오 접속
- 마크베이스는 127.0.0.1 5656 및 (HTTP:5654)에 서비스 가능함.

위의 내용에 대해 궁금한 점은 물어보고, 데모 구현 계획을 세워라.
